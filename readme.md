# App Remedios Chile

Aplicación web para consultar, comparar y gestionar información sobre remedios en Chile.

## Objetivo

Desarrollar una plataforma que permita a los usuarios buscar medicamentos, revisar precios, disponibilidad, información básica y eventualmente comparar alternativas.

## Estructura

- frontend: interfaz de usuario
- backend: API REST y lógica del negocio
- docs: documentación del proyecto

## Stack

- React
- Node.js
- Express
- PostgreSQL
